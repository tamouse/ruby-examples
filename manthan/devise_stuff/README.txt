README
======

Example rails application to show setting up devise to allow users to
use a username or an email to login. Show how it can work with
forgotten passwords and confirmation emails as well.

User nahtam on IRC complains that they can't get this to work, so I'm
trying to see if I can make it work.

Database: Postgresql 9.2.4
Rails: 4.0.3
Ruby: 2.0.0p353
Devise: 3.2.2
